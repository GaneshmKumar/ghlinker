# ghlinker
Github profile linker

# Dependencies
* Bootstrap
* Jquery
* Font awesome

# How to use
* Download the `ghlinker.css` and 'ghlinker.js' files from the repository
* Link them to your HTML page
* you can import the profile by calling the function
  `$.link(<github_username>);`
* Example
  `$.link('ganeshmkumar');
